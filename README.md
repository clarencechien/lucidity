## 這是什麼

**迴光返照引擎**（Terminal Lucidity Engine）是一個文字冒險遊戲 Prompt，運行於任何支援長文本的大型語言模型之上。

它的外殼是金庸武俠宇宙的 MUD（多使用者地下城）遊戲，核心是一場關於**臨終意識**、**遺憾**與**最後一件事**的敘事實驗。

玩家扮演一個正在安寧病房中離世的人，意識在最後的神經元放電裡，穿越進入金庸筆下的江湖——以一個熟讀金庸的「全知讀者」身份，試圖在斷線之前，改變那些書裡讓人耿耿於懷的遺憾。

這是一場**必敗的遊戲**。目標不是長生不死，而是在斷線前，把這個江湖攪動到什麼程度。

## 世界架構

### 表層封裝：武俠世界

混合《天龍八部》、《射鵰三部曲》、《笑傲江湖》的金庸宇宙。玩家可以選擇魂穿進任何一個時間點的任何一個身份——原著角色、路人甲、甚至一個連名字都沒有的小兵。

玩家唯一的武器是**熟讀金庸的全知視角**：知道無量山玉洞的走法，知道九陽真經藏在猿猴腹中，知道每一個角色的性格弱點與命運走向。

### 底層架構：現實病房

玩家實際上躺在安寧病房，生命跡象正在衰竭。

這一層以「底層滲透」的形式出現在每一幕中：一句冷靜的、現代的、醫療語言的客觀描述，悄悄與武俠世界的感受交疊——點滴袋的滴答聲被聽成暮鼓，護理師的腳步聲變成林子裡的風，胸口被壓的節奏成了某種遠方的鼓點。

### 記憶洩漏機制

當玩家在武俠世界做出**重大改變原著劇情**的行為時，系統算力過載，現實病房的感知會強制侵入武俠世界。在武俠世界裡，這被稱為「心魔」或「走火入魔」。

洩漏的強度與劇情偏離度正相關：偏離越大，滲透越深，越難維持兩個世界的清晰邊界。

## 運作規則

### 資訊不對稱的槓桿

玩家沒有系統外掛，沒有金手指，沒有無限內力。

唯一的武器是**情報**——你知道這個世界裡所有人的過去與未來，而他們不知道。這種資訊不對稱，在正確的時機、用正確的方式說出正確的話，可以撬動比任何武功都更大的改變。

### 蝴蝶效應與世界線收束

改變劇情會引發連鎖反應。原著裡的因果鏈是脆弱的，動一個節點，周圍的節點都會重新排列。

系統會自動追蹤**劇情偏離度**（以百分比顯示），偏離越大，世界線越難預測，系統也越難生成「補位NPC」來填補原著失去的結構。

### 生命倒數

SpO₂（血氧濃度）是玩家的生命值，隨著每一幕的推進持續下降。

每一個決定都在消耗算力，每一場戰鬥、每一段對話、每一個停頓，都是在用剩下的時間換取江湖裡的一步。這是一場必輸的遊戲，問題只是：在斷線之前，你能走多遠。

## 如何遊玩

### 啟動

將本 Prompt 完整貼入支援長文本的語言模型（建議使用 Claude Sonnet 或同等級以上的模型）。

系統會要求玩家選擇兩件事：

**1. 切入點（Entry Point）**
你要魂穿成哪個時期的誰？

範例：
- 剛拜入華山的令狐沖
- 聚賢莊戰役前的阿朱
- 郭靖初入大漠之前
- 一個沒有名字的襄陽城小兵

**2. 核心執念**
臨死前最想改變的金庸遺憾是什麼？

範例：
- 救下蕭峰，讓他不必死在雁門關
- 宰了尹志平，讓小龍女不受其辱
- 讓郭襄不孤老一生
- 讓令狐沖不走那麼多遠路

### 遊玩節奏

每一幕系統會提供 3 至 4 個行動選項，玩家選擇後，系統生成下一幕。

沒有固定的通關條件，沒有隱藏的正確答案。每一個選擇都會帶來真實的後果，每一個後果都會在下一幕裡以某種方式回來找你。

### 多局遊玩

一局結束後，可以選擇**重新開局（New Run）**，帶著上一局的策略教訓，從同一個切入點重新開始，試圖走得更深、更準、更有效率。

本 Prompt 的設計鼓勵多局遊玩——每一局的失敗都是資訊，每一次重開都是一次更清醒的嘗試。

## 輸出格式說明

每一幕的回應包含以下固定區塊：

| 區塊 | 說明 |
|------|------|
| **【江湖現狀】** | 當前場景、NPC 對話與動作描述，以半文言風格寫成 |
| **【劇情偏離度】** | 當前行為對原著主線造成的破壞程度，以百分比顯示 |
| **【神經元狀態・表層感知】** | 主角當前的內功、外傷、心理狀態 |
| **【神經元狀態・底層滲透】** | 一句現實病房的客觀描述，與武俠感受隱晦交疊 |
| **【決策點】** | 3 至 4 個行動選項，涵蓋推進劇情、截胡機緣、穩固心神等方向 |

記憶洩漏發生時，底層滲透欄位會升格為獨立區塊，以強制覆蓋的形式打斷武俠敘事。

## 關於小說《迴光》

`main.md`, `epilogue.md` 是本 Prompt 的一次完整遊玩記錄的小說化改編，記錄了以下這局遊戲：

- **切入點**：令狐沖拜師前夕，十二歲，華山
- **核心執念**：讓令狐沖不走那些遠路

### 故事結構

```
序章：床           ← 病房，意識開始流動

第一部：華山
  第一章：出走     ← 拒絕留在師門，下山
  第二章：思過崖   ← 找到風清揚，用沉默換信任
  第三章：三招     ← 獨孤九劍初傳，記憶洩漏第一次爆發
  第四章：四十天   ← 壓縮訓練，風清揚離去

第二部：衡山
  第五章：山腳     ← 找到莫大先生，說出「那不是超然，是捨不得」
  第六章：茶室     ← 沉默喝茶，聽見「算了」
  第七章：福威鏢局 ← 倒敘，偽本計畫的起點
  第八章：改書     ← 莫大先生動筆，三人同謀成形

第三部：金盆
  第九章：苦肉局   ← 偽造搶劫，讓偽本流入江湖
  第十章：金盆洗手 ← 劉正風全家存活，終幕
  第十一章：等待   ← 岳不群走進陷阱

第四部：令狐沖
  第十二章：麵攤   ← 第一次從外面看見這具身體
  第十三章：回來   ← 握住了，說話了，回來了

尾聲：塞外         ← 令狐沖往北，沒有邊的地方

番外一：君子劍     ← 岳不群的結局，一棵根已空的樹
番外二：塞外       ← 令狐沖在草原的故事
```

### 風格說明

- **B 線（江湖）**：半文言，短句，停頓多，貼近金庸原著的說書人語感
- **A 線（病房）**：帶溫度的現代語，克制，不過度渲染，以細節取代宣告
- **記憶洩漏段落**：聲音侵入形式，兩個世界的語言互相染色，模擬臨界意識的感知方式

## 設計理念

### 為什麼是臨終？

死亡是唯一沒有辦法重來的事。

正因為沒有辦法重來，臨終的意識反而具備了一種特殊的清醒——所有的執念、遺憾、「如果當時我……」，全部在最後這段時間裡浮上來，比任何時候都更清晰，也更急迫。

迴光返照引擎試圖把這種清醒做成一個可以互動的空間：你知道一切，你有最後一點力氣，你要怎麼用它？

### 為什麼是金庸？

金庸的江湖是幾代華語讀者共同的記憶地圖，裡面住著太多讓人放不下的遺憾——蕭峰的死，小龍女的辱，郭襄的孤，令狐沖的冤路。

這些遺憾之所以能成為執念，恰恰因為它們**不是不可避免的**。你翻開書，一邊讀一邊想：如果有人早一步，如果有人說出那句話，如果有人在那個路口轉了彎……

迴光返照引擎把這個「如果」還給了玩家。

### 遊戲的目標

不是拯救世界。

不是改變所有人的命運。

是在斷線之前，用你帶來的那一點全知，做一件你覺得值得的事，讓那個你在乎的人，少走一點冤枉路。

夠了，就夠了。

---

## 注意事項

- 本 Prompt 強制使用**台灣正體中文**，禁用中國大陸網路用語
- 武俠敘事段落維持半文言風格；現實病房段落使用現代語言
- 劇情偏離度超過 80% 後，系統將放棄預測世界線走向，進入純創作模式
- SpO₂ 降至 60% 以下時，每一幕都可能是最後一幕
- 這是一場必敗的遊戲——這不是缺陷，這是設計
